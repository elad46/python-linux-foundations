# Linux File Operations

Linux provides simple commands to manage files and directories.

## cp (copy)
Used to copy files and directories.
cp file1.txt file2.txt
cp -r dir1/ dir2/

## mv (move)
Used to move or rename files.
mv old.txt new.txt
mv file.txt /tmp/

## rm (remove)
Used to delete files and directories.
rm file.txt
rm -r directory/
